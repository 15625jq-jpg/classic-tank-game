# 经典坦克大战平衡版

正式发布目录，包含首页、游玩入口、版本更新页与历史版本入口。

在线游玩网址：[https://15625jq-jpg.github.io/classic-tank-game/](https://15625jq-jpg.github.io/classic-tank-game/)

## 结构说明

- `index.html`：正式首页，展示当前版本、更新入口和历史版本入口。
- `play/`：正式游玩入口目录，`play/index.html` 默认跳转到当前正式版本。
- `updates/`：版本更新页目录，展示近期更新摘要。
- `versions/`：历史版本页目录，展示仍可访问的旧版本入口。
- `records/`：完整记录页目录，负责以更适合阅读的样式展示版本说明。
- `更新发布.ps1`：本地自动更新脚本。
- `经典坦克大战平衡版版本调整说明.md`：完整版本调整记录。
